Detailed Description

template<class worker>
class assistant< worker >

An assistant for running workers.

Creates a socket, and handles sending pings to avoid timeouts.

The worker class must have a public constructor, a member std::string const service_name and a method operator()(msg::request && request) -> std::vector<zmq::message_t>. See datastore for an example.

template<class worker >
auto assistant< worker >::run ( ) -> void
inline

Run the worker for one iteration.

The assistant will first try to accept a message from the broker. If the message is a request then it will be passed to the worker, otherwise it will be handled by the assistant. Depending on the message, the assistant may or may not send a message in response.
