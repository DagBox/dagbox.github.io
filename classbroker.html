Create a message broker.

Parameters
ctxThe 0MQ context the broker should run in.
addrThe address the broker should bind to. All workers must connect to this address.
worker_timeoutThe time in miliseconds after which the broker should consider a worker dead if there hasn't been any response. The workers should be configured with a worker timeout that is less than this number. This number should be larger than the time it would reasonably take for a request to complete.

auto broker::run ( ) -> void

Run the message broker for one iteration.

This function should be called repeatedly to run the broker. When called, if there are no messages to be recieved, it will wait for some time.

Instead of calling this function directly, consider using component to run the broker.
